Setup Firebase Diperlukan

Ikuti 6 langkah berikut untuk menghubungkan website ke Firebase:

1
Buat Project Firebase

Buka console.firebase.google.com → Add project → beri nama (contoh: kuis-jangka-sorong) → Create project

2
Aktifkan Realtime Database

Sidebar → Build → Realtime Database → Create Database → lokasi asia-southeast1 → pilih Test mode → Enable

3
Dapatkan Config Web App

Project Settings (⚙️) → Your apps → klik ikon </> → Register app → copy seluruh firebaseConfig

4
Edit index.html

Buka file ini dengan Notepad/VS Code → cari teks GANTI_API_KEY → ganti seluruh blok firebaseConfig dengan yang kamu copy

5
Atur Rules Database

Firebase → Realtime Database → tab Rules → paste dan Publish:

{
  "rules": {
    "responden": {
      ".read": true,
      ".write": true
    }
  }
}
6
Upload ke GitHub Pages

Upload index.html, style.css, app.js ke repo GitHub → Settings → Pages → main/root → Save

✅ Setelah semua langkah selesai, refresh halaman ini!
