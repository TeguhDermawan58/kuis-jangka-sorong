Setup Firebase Diperlukan

Ikuti langkah berikut untuk menghubungkan website dengan Firebase:

1
Buat Project Firebase

Pergi ke console.firebase.google.com → klik Add project → beri nama (contoh: kuis-jangka-sorong)

2
Aktifkan Realtime Database

Di sidebar kiri → Build → Realtime Database → Create Database → pilih lokasi asia-southeast1 → pilih mode Test mode (untuk awal)

3
Dapatkan Konfigurasi

Project Settings (⚙️) → Your apps → klik ikon Web (</>) → Register app → copy firebaseConfig

4
Edit file index.html

Ganti bagian firebaseConfig di dalam tag <script type="module"> dengan config yang kamu copy

5
Atur Rules Database

Di Realtime Database → tab Rules → paste rules berikut lalu klik Publish:

{
  "rules": {
    "responden": {
      ".read": true,
      ".write": true
    }
  }
}
6
Upload ke GitHub & Aktifkan Pages

Push semua file ke repository GitHub → Settings → Pages → Source: main / root → Save

💡 Setelah setup selesai, refresh halaman ini dan website siap digunakan!
